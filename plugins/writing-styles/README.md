# writing-styles

A Copilot CLI plugin that rewrites text in different voices.

## Skills

- **biz** — Rewrites text using formal business language.
- **ilya-voice** — Rewrites text to match Ilya Fainberg's personal email style.

## Usage

After `copilot plugin install writing-styles`, ask Copilot things like:

> Rewrite this paragraph in Ilya's voice: ...

> Make this email sound more formal/business-like.

The matching skill is invoked automatically based on the request.
